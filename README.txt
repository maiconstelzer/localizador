+++ Sobre +++  
Gera páginas comumente utilizadas por usuários solicitando sua respectiva localização, ao usuário aceitar a mesma é repassada na API com suas respectivas coordenadas

+++ Dependências +++  
Debian 12: php-fem, python3, python3.requests, python3.packaging, python3.psutil

+++ Observações +++  
Acaso usuário se encontra em um PC/Notebook que não possua placa de rede wifi ou gps, as coordenadas retornadas podem ser do provedor de internet. Em smartphones pode acontecer do usuário utilizar uma aplicação de GPS FAKE, exemplo: Fake Traveler, sendo que as coordenadas retornadas podem não condizer com o esperado.
